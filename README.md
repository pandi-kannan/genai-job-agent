# 🎯 AI Job Application Assistant

An autonomous AI agent that helps job seekers evaluate and tailor their resumes for specific job postings — calculating a fit score, analyzing skill gaps, researching companies, and generating quantified resume bullet points, all through a multi-step LangGraph agent pipeline.

🔗 **Live App:** [your-streamlit-app-url-here](https://genai-job-agent-dgbhv4jzjnmrrc63zkqyki.streamlit.app/)

## 📌 Problem Statement

Job seekers, especially freshers, spend significant time manually comparing their resume against job descriptions, judging whether they're a good fit, researching companies, and rewriting bullet points for every application. This project automates that workflow using an AI agent that reasons through the process step by step, rather than relying on a single prompt.

## ✨ Features

- **Fit Score** — instantly scores how well a resume matches a specific job (0-100) with a clear label (Strong/Good/Moderate/Weak Match) and reasoning
- **Job Description Summarization** — extracts key requirements and responsibilities from any job posting
- **Resume Gap Analysis** — compares your resume against the job requirements and identifies specific skill/experience gaps
- **Live Company Research** — pulls up-to-date company information via web search
- **AI-Generated Resume Bullets** — produces tailored, quantified bullet points grounded in your actual resume content
- **Application History** — every analysis is saved to a PostgreSQL database, so you can track past applications and revisit tailored suggestions

## 🏗️ Architecture

The core of this project is an **agent graph** built with LangGraph, where each step is an independent node that passes shared state forward:

[Job Description + Resume + Company Name]
↓
Summarize Job Requirements
↓
Analyze Resume Gaps
↓
Calculate Fit Score
↓
Search Company Info
↓
Generate Tailored Bullets
↓
[Results + Saved to DB]

Unlike a simple linear script, this graph structure allows for conditional logic and clear separation of concerns — each node can be modified, tested, or extended independently (e.g., adding a new tool or a conditional branch for low-fit roles).

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|

## ⚙️ Setup & Installation

1. Clone the repository
```bash
git clone https://github.com/pandi-kannan/genai-job-agent.git
cd genai-job-agent
```

2. Create a virtual environment and install dependencies
```bash
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

3. Set up environment variables — create a `.env` file:
GROQ_API_KEY=your_groq_api_key
DATABASE_URL=your_postgresql_connection_string

4. Initialize the database
```bash
python db.py
```

5. Run the app
```bash
streamlit run app.py
```

## ⚠️ Note on AI-Generated Content

The Fit Score, gap analysis, and resume bullets are generated by an LLM based on the resume and job description provided. While the agent is prompted to stay grounded in the candidate's actual resume content, any specific metrics or numbers suggested in the resume bullets should be verified and adjusted by the user before use — they are AI-generated suggestions, not verified claims.
